# Contributing to Native PDF Ink

Thanks for your interest in contributing.

## Development setup

1. Clone the repository.
2. Install dependencies using the project's package manager.
3. Run the local test or lint commands before opening a pull request.

## Pull requests

- Keep changes focused and explain the motivation in the PR description.
- Update documentation when behavior changes.
- Add or update tests when the project already has them.

## Code of conduct

This project follows [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
