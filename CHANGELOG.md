# Changelog

## Unreleased

- Added a Draw checkbox, off by default, so Sidecar Apple Pencil input remains normal mouse-like PDF input until annotation is explicitly enabled.
- Matched annotation controls to Obsidian's compact PDF-toolbar icons and isolated save status in a fixed-width area.
- Replaced whole-stroke deletion with a circular eraser that preserves untouched stroke segments.
- Added a scale-aware circular cursor and streamlined Small, Medium, and Large size choices.

## 0.1.0 — 2026-07-12

- Added local-first handwriting and annotation for Obsidian PDF views.
- Added versioned sidecars, autosave, recovery, and separate annotated-PDF export.
- Added optional confirmed direct-PDF writes with backups enabled by default.
- Added desktop and mobile input adapters, shared tools, and accessible settings.
